Resonance —
Violin Orientation Monitoring System

An embedded sensing system that monitors violin orientation in real time — combining an IMU, microcontroller, 3D-printed housing, and browser-based dashboard to support musicians in developing proper technique.

Embedded IMU / Sensors RP2040 3D Printing Web App User Testing

1. The Idea

Overview

Violin technique depends heavily on instrument orientation — the angle at which the violin is held affects tone, bowing mechanics, and long-term posture. Resonance was designed to give players real-time feedback on orientation using an inertial measurement unit (IMU) mounted directly on the instrument.

Built as a project at the Segal Design Institute at Northwestern, the system went through the full design cycle: user research, prototyping, hardware integration, firmware development, and user testing with real musicians.

3. Software & Design Process

The Dashboard

Orientation data from the IMU is streamed to a browser-based dashboard that visualizes the violin's position in real time. The interface was designed with musicians in mind — intuitive enough to use mid-practice without technical overhead.

  • Integrated Adafruit Feather RP2040 with 9-DoF IMU for real-time motion tracking
  • Developed browser-based dashboard to visualize orientation data during user testing
  • Conducted iterative prototyping and user-centered design evaluation with violinists
  • Designed and fabricated 3D-printed housing compatible with standard shoulder rests
